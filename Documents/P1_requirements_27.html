Purpose and Goals

The purpose of the website is to provide color schemes based upon a user given color.

This will be accomplished by having a input field on the home page, where the user can enter the color they intend to use for the scheme. The colors will then be generated by javascript on the backend, and thereafter be shown as div's with a background color on the website. The div's will have input fields (for easy copying), that is read only, showing the hex value of the colors. The website will be quite simple. However, the design should be quite adequate.

Intended Audience

The intended audience for the website is designers, UI developers, and artists. Because of the intended audience, the graphics and design of the website should be good-looking. Furthermore, if users find the site easy to use, they will be more likely to return to it.

Content of site

Home page is a basic input field and a button. The main page should also have a nav bar and some nice artwork in the background. Upon clicking the button, the backend should generate some color schemes, and add classes to html elements, to show it's content. Therefore, the content of the home page expands after generating the color schemes. This will of course be applied to mobile users as well.

Requirements

Functional Requirements

  • The website should accept user input of a color.
  • The user should be able to navigate to subpages.
  • The user should be able to copy the hex codes of the generated colors.

Non-functional Requirements

  • The generator must load in less than 1 second
  • The website should load in less than 2 seconds
  • Mobile users should be able to navigate anywhere on the page in 2 clicks. However, PC users should be able to do the same in one click.
