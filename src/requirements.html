The purpose of the website is to provide color schemes based around a user given color.

This will be accomplished by having a input field on the home page, where the user can enter the color they intend to use for the scheme. The colors will then be generated by javascript on the backend, and thereafter be shown as div's with a background color on the website. The div's will have input fields (for easy copying), that is read only, showing the hex value of the colors.

The intended audience for the website is designers, UI developers, artists
